## 📄 My Resume - HTML Capstone Project

This project is part of my learning journey and was created as a practice exercise while taking a web development course. The goal of this project was to build a simple and structured resume page using only HTML.

🔗 Live Demo:
https://jamesaamars.github.io/My-Resume-capstone-1-/

---

## 🚀 About The Project

In this project, I designed and developed a personal resume webpage using pure HTML. The main focus was on:

* Structuring content with semantic HTML
* Organizing sections like education, experience, and skills
* Creating a clean and readable layout without using CSS or JavaScript

---

## 🛠️ Built With

* HTML5

## 🎯 What I Learned

* Understanding the importance of content hierarchy
